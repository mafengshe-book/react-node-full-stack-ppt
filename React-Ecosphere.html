服务器端渲染(SSR)

为什么使用服务器端渲染(SSR)？

与传统 SPA（Single-Page Application - 单页应用程序）相比，服务器端渲染(SSR)的优势主要在于：

  1. 更好的 SEO，由于搜索引擎爬虫抓取工具可以直接查看完全渲染的页面。
  2. 解决首屏白屏问题
  3. 顺应时代潮流（开玩笑啦！）

核心函数

renderToString 方法把Virtual DOM转换成HTML字符串。从而达到服务端渲染的目的

import { renderToString } from 'react-dom/server';
